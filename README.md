Azure AD B2C - Angularjs sample (Web and Mobile) app
=====================

A starting project for Ionic that optionally supports using custom SCSS.

## Project set up:
npm install
bower install

## AD set up:
ADAL-B2C configuration

## try the sample:
Web App:
ionic serve

Mobile App:
cordova platform add android
cordova build android

## Code explanation:
Web App
Mobile APp
